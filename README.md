# DecodeLabs-Data-Science-Intership

## Overview
This project was completed as part of the DecodeLabs Data Science Internship program.

The project focuses on:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insights

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
The dataset contains customer orders, product sales, payment methods, and order status information.

## Key Insights
- Identified top-selling products
- Analyzed customer payment methods
- Visualized sales trends
- Explored customer purchasing behavior
